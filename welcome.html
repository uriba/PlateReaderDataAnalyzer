Welcome to the Plate Reader Data Analyzer site. The aim of this site is to facilitate the analysis of data measured using plate readers. The site currently supports .xlsx files generated by Tecan readers.

The site implements algorithms to calculate growth rates and doubling times based on Optical Densitiy (OD) measurements. Analysis of fluorescnet and luminescent reporter genes to deduce protein level, promoter activity and other related quantities is planned to be implemented as well.

To begin, upload a plate-reader generated file using the field at the left side of the page. Once a file is uploaded, and according to the analysis type, various controls will appear on the left side of the page allowing adjustments of parameters for the analysis. The main page (where currently this text is displayed) will present graphs showing the results of the analysis.

As plate reader data tends to be of high volume and complexity, the processing and display of data may take a few seconds, up to about a minute, to appear (for 96-well plates and ~ 100 measurements per well). Focusing the analysis on specific rows, columns, or wells will reduce the amount of data processing, thus making the site speedier.

Future plans for the site include (by no particular order): Interactive (zoomable, pannable, and with value highlighting upon hovering) graphs. Export of analyzed data in table format. Analysis of various reporter genes related statistics, promoter activity, protein level etc. Supporting wider variety of input files from different plate readers, upon request.
